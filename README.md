# 🎭 Гайд по созданию персонажа на Frendi AI

Полный интерактивный веб-гайд по созданию персонажей на платформе Frendi AI — от идеи до живого бота.

**Автор:** Матти/Matte (matovya)
- [Профиль на Frendi](https://frendi.ai/user/455adeea-33b7-44a2-a640-89bff5eefd20)
- [Telegram: «Хрущевка frendi.ai»](https://t.me/matovya_creator)

## ✨ Особенности

- 📚 17 подробных разделов с примерами и практикой
- 🌍 Переводы всех примеров — английский + русский
- 💬 Интерактивные подсказки с переводами терминов
- 🎨 5 уникальных иллюстраций
- 📱 Полностью адаптивный дизайн
- 🌑 Темная тема
- ⚡ Single-file приложение (~94KB gzip)

## 🚀 Деплой на GitHub Pages

1. Создайте публичный репозиторий на GitHub
2. Загрузите все файлы проекта
3. Перейдите в Settings → Pages
4. В разделе Source выберите **GitHub Actions**
5. Готово! Сайт автоматически соберется и будет доступен

## 💻 Локальная разработка

```bash
# Установка зависимостей
npm install

# Запуск dev сервера
npm run dev

# Сборка для продакшена
npm run build
```
